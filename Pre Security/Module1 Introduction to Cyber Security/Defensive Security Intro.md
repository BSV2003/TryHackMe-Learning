# Defensive Security Intro
Introducing defensive security and related topics, such as Threat Intelligence, SOC, DFIR, Malware Analysis, and SIEM.

## What is Defensive Security?

Defensive Security focuses on **protecting systems, networks, and data** from cyber attacks.  
It works as the counterpart to offensive security.

The two primary objectives of defensive security are:
- **Preventing intrusions** from occurring
- **Detecting intrusions** when they occur and responding effectively

Defensive security is mainly handled by **Blue Teams**.

### Relationship with Offensive Security

In the previous lesson, offensive security focused on:
- Exploiting software vulnerabilities
- Leveraging insecure configurations
- Abusing weak or unenforced access control policies

These activities are typically performed by **Red Teams** and **Penetration Testers**.

Defensive security uses the knowledge of these attack techniques to **build stronger defenses**.
